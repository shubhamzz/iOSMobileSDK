Requirements

In order to integrate the QualarooMobileSDK into a 3rd-party app, the app must satisfy the following requirements:

  • Minimum deployment target set to iOS 8.0 or later
  • Xcode 6 or later

Notes on Code Style

All the source code written in Objective-C / C in this project is formatted using clang-format. In order to automatically format code when saving a file in Xcode, please make sure you follow the next steps:

  1. Have Alcatraz installed in Xcode (see installation steps on their website.)
  2. Have the plugin BBUncrustifyPlugin installed with Formatter set to Clang, Perform Xcode syntax-aware indenting disabled, Clang Style set to Custom Style (File), and Format on Save enabled.
  3. Or, alternatively, have the plugin ClangFormat installed (at the time of writing, this plugin did not work correctly with Xcode 7.3.)

To modify the rules used to format the source code, please modify .clang-format.
